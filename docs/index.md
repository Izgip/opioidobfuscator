# Leaph - VM Obfuscator Learning Project

Educational implementation of a Virtual Machine-based obfuscator for Luau.

## About
This project demonstrates VM architecture for code obfuscation, created as a learning resource. 

## Repository
[View source code](https://github.com/Izgip/leaph)

## Purpose
- Learn VM design patterns
- Understand bytecode execution
- Study multi-layer encryption
- Explore anti-debugging techniques

## Note
This is shared as educational material, not production code.
